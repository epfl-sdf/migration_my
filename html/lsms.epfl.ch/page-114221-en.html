      Implementation of Dynamics for Bernoulli beam elements within Akantu

      Bachelor Project - Spring 2014

      The purpose of this project is to implement dynamics resolution for Bernoulli beams within a finite element library (Akantu). The work consists in adding the dynamic model to the structural mechanics branch of the library and to validate it by running tests on cases which analytical solution are known in order to compare the results. Another test is performed on a more complex problem to verify the behavior of the program when confronted to a large number of elements.

      The parallel objective of the project is to simultaneously learn programming in C++ and acquire the necessary notions in the finite element method (FEM) and structural dynamics.

      The report details the implementation of the model as well as the theoretical notions and the different steps toward its completion.
