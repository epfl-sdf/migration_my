          Orbital Debris Removal Satellite : Clean-mE

          List of Semester / Master projects for Autumn 2011, please contact Mrs. Muriel Richard for keeping informed of new projects for Autumn 2012.

          The Swiss Space Center is starting the design and development of a nanosatellite (30 x 30 x 30 cm), which goal will be to remove orbital debris in low-Earth orbit. Although at first sight that may seem simple to do, it is in fact a very difficult task, especially if one wants to remove more than one debris. Several new technologies will kick in the design, and several partnerships with industry are being established. All disciplines are needed. See below for specific topics, but if you do not fit within these topics, and are still interested in participating, contact us (muriel.richard@epfl.ch).

          Attitude determination and Control System for a Space Debris Removal Spacecraft

          Project Type: Semester / Minor / Master project
          Background: MT/GM, Robotics, Control Systems, Minor in Space Technologies, Other

          The determination of the attitude is essential for the success of the CleanMe project. Unlike for conventional spacecraft, CleanMe’s mass distribution and inertia can drastically change during the mission and cannot be determined in advance. Furthermore, the grasping maneuver might heavily perturb the spacecraft attitude. A very powerful, flexible and adaptive control system is therefore required. This is a completely new challenge in space flight. 

          This project is in cooperation with the TOPO Lab.
          Point of contact: Muriel Richard

          Mission Design for a Space Debris Removal Spacecraft

          Project Type: Semester / Minor
          Background: MT/GM, Robotics, Control Systems, Minor in Space Technologies, Other

          The trajectory of the space removal spacecraft has to be optimized in order to minimize fuel consumption while maximizing the number of captured debris. Using ESA’s MASTER model, Satellite Toolkit and MATLAB the most interesting zones for debris removal shall be identified. Then the trajectory the required maneuvers and the fuel consumption for the mission shall be assessed.

          Point of contact: Muriel Richard

          Mechanical Arm and Grabbing System for a Space Debris Removal Spacecraft

          Project Type: Semester / Minor / Master project
          Background:  MT/GM, Robotics, Control Systems, Minor in Space Technologies, Other

          This topic will start the design of a simple mechanical device which will grasp the debris to either store it or send it down to Earth. Several concepts for robotic arm have been proposed and evaluated, and the student will start by understanding the pros and cons of each. Following a list of constraints related to space, nanosatellites and the interaction between the debris and Clean-mE, the student will design the arm. If chosen for a master project, the student will built a prototype for demonstration. Structural analysis will need to be performed as well, with simulations. If motivated, this design will then be tested during a parabolic flight. 

          Point de Contact: Muriel Richard, ELD 014

          Vision System for a Space Debris Removal Spacecraft

          Project Type: Semester / Minor / Master project
          Background:  MT, GM, IN, I&C, MA, others

          For the rendezvous maneuver and the grasping of the debris by the CleanMe spacecraft, a stereo-vision system is required. This system will be used to command the grasping mechanism as well as the approach maneuver. Furthermore it has to recognize the rotation rate and the volume of the debris.

          This project is in cooperation with the CV Lab
          Point de Contact: Muriel Richard

          Modeling of a MEMS based Earth Sensor

          Project Type: Semester / Minor / Master project
          Background:  ME, STI, EL, others

          The Microsystems for Space Technologies Laboratory of EPFL is developing a novel MEMS based Earth Sensor. Current state of the art Earth Sensors work by imaging the Earth`s horizon in IR. This requires optical access to external satellite faces, and multiple units so that the Earth Vector can be obtained no matter where the satellite is oriented. Each of these units need to be thermally controlled and shielded against radiation.

          The MEMS based sensor aims to do away with these constraints by enabling a single compact instrument that does not use optics, can be positioned flexibly within the satellite, and due to MEMS technology, uses lower power and has less mass. The MEMS Earth Sensor works by sensing the displacement of its proof mass due to the Gravity Gradient Torque. This displacement depends on the angle of the MEMS chip to the Earth Vector. By sensing this displacement by capacitive means the attitude of the satellite can be determined. Currently, a COTS chip is used to measure this capacitance, and the electronics are a source of noise, and therefore error.

          The performance of this sensor depends on numerous other factors, such as temperature, environmental damping, satellite altitude, and accuracy with which the altitude is known. Since it is an inertial sensor, external factors such as the moons gravity, satellite spin and distance from center of spin are possible sources of error to the MEMS Earth Sensor. The MEMS chip has secondary inertial sensors that determine if the satellite is spinning and accordingly provide an error correction.

          You will develop a model for the MEMS Earth Sensor in software such as MATLAB or similar, that takes into account all these parameters and provides an estimate for the performance and error estimates for this sensor for a range of temperatures, satellite altitudes, satellite spin rates, etc. The model needs to be scalable in order to easily allow additional parameters to be added in future.

          Number of students: 1

          Point de Contact: Kaustav Ghose; e-mail: kaustav.ghose@epfl.ch
          Herbert Shea; e-mail: herbert.shea@epfl.ch
